About

The aim of this project is to make Covid-19 testing simpler and easy for common people, and to reduce the overhelming burden on the Health-care workers. We train Machine learning models on the dataset containing the common symptoms and corresponding test results, provided by the Health Ministry of Israel.
The model calculates the accuracy scores for different Ml algorithms and the chooses the algorithm with highest accuracy score to predict the result for the data, that the user has entered.
